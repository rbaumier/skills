## Discriminating Assertions (28 -- with-skill PASS, without-skill FAIL)

### Timing & Easing (6/6 discriminating)
- **timing-hover-range**: Hover/press feedback 120-180ms (without: says 200-300ms)
- **timing-page-transition-range**: Page transitions 500-800ms (without: says 300-500ms)
- **timing-gpu-only**: Only animate transform and opacity (without: animates background-color)
- **timing-no-linear**: No linear easing for motion (without: not mentioned)
- **timing-custom-bezier**: Custom Bezier curves over defaults (without: uses ease-in-out)
- **timing-scale-range**: Scale 0.95-1.05, never from 0 (without: no rule stated)

### Accessibility (3/3 discriminating)
- **a11y-reduced-motion**: Respects prefers-reduced-motion (without: not mentioned)
- **a11y-no-vestibular**: No excessive vestibular triggers, caps parallax (without: uses 50% parallax)
- **a11y-keyboard**: Keyboard-initiated actions should not animate (without: not mentioned)

### GSAP React (4/5 discriminating)

- **gsap-scope**: Pass scope to useGSAP (without: no scope)
- **gsap-context-safe**: contextSafe() for event handlers (without: not mentioned)

### Motion Gaps (2/5 discriminating)

- **gaps-exit-subtler**: Exit subtler than enter (without: symmetric values)

### Sound Design (4/5 discriminating)

### Audit Workflow (4/4 discriminating)

### Library Selection (2/3 discriminating)

## Compression Recommendations

**Non-discriminating rules** (7) can be compressed to single lines -- LLMs already know these:
- `gsap.registerPlugin()`, AnimatePresence for conditionals, exit animations, tab content animation, sound toggle, GSAP ScrollTrigger recommendation, transform/opacity only

**Discriminating rules** (28) must keep their current detail level -- these are the skill's unique value:
- Specific duration thresholds (120-180ms hover, 500-800ms page)
- Custom Bezier curves with exact values
- Scale range 0.95-1.05
- 3-designer audit framework with context-dependent weighting
- Lenis + GSAP integration requirement
- Enter recipe (opacity + translateY + blur)
- Exit subtler than enter
- Sound: volume 0.3, no decorative, visual equivalent, single AudioContext
- GSAP React: useGSAP hook, scope, contextSafe, 'use client'
- Performance: will-change lifecycle, direct style for drag, velocity swipe
