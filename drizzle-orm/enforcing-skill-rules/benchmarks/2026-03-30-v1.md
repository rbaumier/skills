## Key Differentiators

The skill enforces patterns that general knowledge misses:

1. **$type<T>() for JSON** — Without skill, JSON columns are left as `unknown` and cast with `as` at read time. The skill ensures compile-time type safety.
2. **Cursor pagination** — Without skill, OFFSET pagination is used even when the prompt explicitly mentions 10M rows.
3. **Serverless connection reuse** — Without skill, a new Pool is created per invocation with no caching.
4. **Column selection** — Without skill, `select()` fetches all columns including large text fields for 500K rows.
5. **satisfies Config** — A subtle but important TypeScript pattern the skill enforces.
