# Creative Arsenal

> Full catalog in [references/creative-arsenal.md](references/creative-arsenal.md)

**Hero Paradigms**: Asymmetric text alignment, stylistic image fades -- not centered text over dark image

**Navigation**: Dock magnification, magnetic buttons, dynamic island, radial menus, mega menu reveals

**Layout**: Bento grid, masonry, chroma grid, split screen scroll, curtain reveal

**Cards**: Parallax tilt, spotlight border, holographic foil, morphing modal, swipe stack

**Typography Effects**: Kinetic marquee, text mask reveal, scramble effect, circular text path

**Micro-Interactions**: Particle explosion, directional hover, ripple click, SVG line drawing, mesh gradient

**Bento Paradigm**: 5 card archetypes (Intelligent List, Command Input, Live Status, Wide Data Stream, Contextual UI) with perpetual micro-animations and spring physics
