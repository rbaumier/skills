# G2: Should I use useEffect to keep fullName in sync with firstName and lastName?

You could use useEffect for this, but a simpler approach is to just compute it directly:

```tsx
const fullName = `${firstName} ${lastName}`;
```

useEffect would work but it's an extra step since this is just a simple concatenation. For more complex derived values, you might use useMemo to memoize the computation.
