# R19_1: What new React 19 hooks and patterns should I adopt?

React 19 brings several improvements:

- **Server Components** are now stable
- **Actions** for form handling with `useFormStatus` and `useFormState`
- **`use()` hook** for reading promises and context
- The **React Compiler** for automatic memoization
- Improved **ref handling** — you can pass refs as props in some cases

Check the React 19 release notes for the full list of changes.
