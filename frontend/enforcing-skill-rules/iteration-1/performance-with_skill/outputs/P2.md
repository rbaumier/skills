# P2: Performance is slow with barrel imports from UI library

**No barrel imports from heavy libraries.** Use direct imports instead.

```tsx
// BAD - barrel import pulls in the entire library
import { Button, Icon, Modal } from '@mylib';

// GOOD - direct imports, tree-shakeable
import { Button } from '@mylib/components/Button';
import { Icon } from '@mylib/components/Icon';
import { Modal } from '@mylib/components/Modal';
```

Barrel imports (`index.ts` re-exports) force the bundler to load every module in the barrel, even if you only use one component. This bloats your bundle and slows startup. Direct imports let tree-shaking work properly and reduce initial bundle size significantly.
