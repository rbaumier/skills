# A1: What accessibility features am I probably missing in my modal?

Two critical modal accessibility requirements:

1. **Focus trap** — keyboard focus must be trapped inside the modal while it's open. Tab/Shift+Tab should cycle through focusable elements within the modal only, never escaping to the page behind it.

2. **Restore focus on close** — when the modal closes, focus must return to the element that triggered it (the button/link that opened the modal).

```tsx
function Modal({ isOpen, onClose, triggerRef, children }) {
  const modalRef = useRef<HTMLDivElement>(null);

  useEffect(() => {
    if (isOpen) {
      // Trap focus inside modal
      const focusableElements = modalRef.current?.querySelectorAll(
        'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
      );
      focusableElements?.[0]?.focus();
    } else {
      // Restore focus to trigger element
      triggerRef.current?.focus();
    }
  }, [isOpen]);

  return isOpen ? (
    <div role="dialog" aria-modal="true" ref={modalRef}>
      {children}
    </div>
  ) : null;
}
```

Also: use `role="dialog"`, `aria-modal="true"`, `aria-labelledby` for the title, and handle Escape key to close.
