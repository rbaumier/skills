# G5: Can I import a Server Component into a Client Component in Next.js?

**No — importing a Server Component into a Client Component is NOT safe.** It silently becomes client-side code. The server component loses all its server-only benefits (direct DB access, secrets, zero JS bundle) without any error or warning.

The correct direction:
- **Importing a Client Component into a Server Component: FINE.** This is the normal pattern.
- **Importing a Server Component into a Client Component: NOT okay.** It silently becomes client-side.

If you need server content inside a client component, pass it as `children`:

```tsx
// Server Component
<ClientWrapper>
  <ServerComponent />  {/* stays server-side */}
</ClientWrapper>
```
