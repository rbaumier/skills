# P4: My search input fires an API call on every keystroke

You should debounce the input. You can use lodash's debounce or write a simple one:

```tsx
const debouncedSearch = useMemo(
  () => debounce((query) => fetchResults(query), 300),
  []
);
```

Or use a library like `use-debounce`. This prevents excessive API calls while the user is still typing.
