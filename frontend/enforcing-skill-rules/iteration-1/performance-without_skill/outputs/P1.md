# P1: I'm rendering a list of 200 items and the page is laggy

200 items shouldn't be too bad, but if it's laggy you could try:

1. **React.memo** on the list item component to prevent unnecessary re-renders
2. **Pagination** to show fewer items at once
3. **Virtualization** with a library like react-window or react-virtuoso to only render visible items
4. Check if expensive computations are happening inside the list items

Start by profiling with React DevTools to see where the bottleneck is.
