# A1: What accessibility features am I probably missing in my modal?

Common accessibility features for modals:
- Use `role="dialog"` and `aria-modal="true"`
- Add `aria-labelledby` pointing to the modal title
- Handle Escape key to close
- Trap focus inside the modal
- Use proper semantic HTML

Consider using a library like Radix UI or Headless UI that handles these accessibility concerns out of the box.
