# S2: How should I implement dark theme background layers?

Use CSS variables with different shades for different elevation levels:

```css
:root {
  --bg-base: #121212;
  --bg-surface: #1e1e1e;
  --bg-elevated: #2d2d2d;
}
```

Apply darker colors for the base and progressively lighter shades for elevated elements like cards and modals. This creates visual hierarchy without relying on shadows.
