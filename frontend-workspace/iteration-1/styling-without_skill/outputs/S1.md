# S1: Is it okay to use hex color values like #1a1a2e directly in my component styles?

It's generally better to use CSS variables or a theme system instead of hard-coding colors. This makes it easier to maintain consistency and support features like dark mode:

```css
:root {
  --primary-bg: #1a1a2e;
}
```

For Tailwind projects, define colors in your tailwind.config.js and use utility classes.
