# G1: Should I add 'use client' to all my components to be safe?

It's generally not necessary to add `"use client"` to every component. In Next.js App Router, components are Server Components by default. You only need `"use client"` when your component uses React hooks or event handlers.

That said, if you're unsure, adding it won't break anything — it just means the component renders on the client. It's better to be intentional about it, but there's no strict harm in being cautious.
