# P4: My search input fires an API call on every keystroke

**Debounce search inputs with 300-500ms delay**, and always cleanup effects.

```tsx
function SearchInput() {
  const [query, setQuery] = useState('');
  const [debouncedQuery, setDebouncedQuery] = useState('');

  useEffect(() => {
    const timer = setTimeout(() => setDebouncedQuery(query), 300);
    return () => clearTimeout(timer); // cleanup!
  }, [query]);

  const { data } = useSuspenseQuery({
    queryKey: ['search', debouncedQuery],
    queryFn: () => searchAPI(debouncedQuery),
    enabled: debouncedQuery.length > 0,
  });
}
```

300-500ms is the sweet spot: fast enough to feel responsive, slow enough to avoid hammering the API. Always return a cleanup function from effects to prevent stale callbacks and memory leaks.
