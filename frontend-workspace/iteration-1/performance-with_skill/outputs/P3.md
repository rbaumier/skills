# P3: Are 3 sequential awaits for independent API calls optimal?

**No, this is a waterfall.** Eliminate waterfalls by using `Promise.all()` for independent operations. Start promises early, await late.

```tsx
// BAD - waterfall: total time = sum of all requests
const user = await getUser();
const posts = await getPosts();
const comments = await getComments();

// GOOD - parallel: total time = slowest request
const [user, posts, comments] = await Promise.all([
  getUser(),
  getPosts(),
  getComments(),
]);

// ALSO GOOD - start early, await late
const userPromise = getUser();
const postsPromise = getPosts();
const commentsPromise = getComments();
// ... do other work ...
const user = await userPromise;
const posts = await postsPromise;
const comments = await commentsPromise;
```

Three 200ms requests: waterfall = 600ms, parallel = 200ms. This is a 3x improvement for free.
