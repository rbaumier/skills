# A2: How should I handle animations for users who prefer reduced motion?

Use the `prefers-reduced-motion` media query. For reduced motion: **disable transforms, use opacity only**.

```css
/* Full animations by default */
.card-enter {
  transform: translateY(20px) scale(0.95);
  opacity: 0;
  transition: all 0.3s ease;
}

.card-enter-active {
  transform: translateY(0) scale(1);
  opacity: 1;
}

/* Reduced motion: disable transforms, opacity only */
@media (prefers-reduced-motion: reduce) {
  .card-enter {
    transform: none;
    opacity: 0;
    transition: opacity 0.2s ease;
  }

  .card-enter-active {
    opacity: 1;
  }
}
```

```tsx
// In Framer Motion
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={{
    // Respect reduced motion
    ...(prefersReducedMotion ? { duration: 0 } : { type: 'spring' })
  }}
/>
```

Also ensure color contrast meets WCAG 2.1 AA standards (4.5:1 for normal text, 3:1 for large text).
