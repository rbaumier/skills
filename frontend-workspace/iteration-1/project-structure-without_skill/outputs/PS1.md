# PS1: How should I organize my React project's folder structure?

A common approach is to organize by type:

```
src/
  components/
  hooks/
  utils/
  services/
  types/
  pages/
```

For larger projects, you might organize by feature:

```
src/
  features/
    auth/
    dashboard/
  shared/
    components/
    hooks/
```

There's no single right answer — pick what works for your team and be consistent.
