# A2: How should I handle animations for users who prefer reduced motion?

Use the `prefers-reduced-motion` media query to respect user preferences:

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

This disables or minimizes all animations for users who have enabled "Reduce Motion" in their system settings. You can also use the `useReducedMotion` hook from Framer Motion.
