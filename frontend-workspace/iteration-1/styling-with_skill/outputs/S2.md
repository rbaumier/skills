# S2: How should I implement dark theme background layers?

Use a **layered background system**: `bg-neutral-bg1/bg2/bg3` for depth, `border-border-subtle` for borders, and **glass classes via `@layer`** for glassmorphism effects.

```css
/* Design tokens */
:root {
  --bg1: #0a0a0f;     /* deepest layer - page background */
  --bg2: #12121a;     /* mid layer - cards, panels */
  --bg3: #1a1a28;     /* surface layer - elevated elements */
  --border-subtle: rgba(255, 255, 255, 0.06);
}

/* Glass classes via @layer */
@layer components {
  .glass {
    background: rgba(255, 255, 255, 0.03);
    backdrop-filter: blur(12px);
    border: 1px solid var(--border-subtle);
  }

  .glass-elevated {
    background: rgba(255, 255, 255, 0.06);
    backdrop-filter: blur(16px);
    border: 1px solid rgba(255, 255, 255, 0.1);
  }
}
```

```tsx
<div className="bg-neutral-bg1">                         {/* page */}
  <aside className="bg-neutral-bg2 border-r border-border-subtle"> {/* sidebar */}
    <div className="bg-neutral-bg3 glass">               {/* card */}
    </div>
  </aside>
</div>
```

Three layers create visual depth without relying on shadows (which are invisible on dark backgrounds).
